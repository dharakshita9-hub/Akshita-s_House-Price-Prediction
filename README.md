Akshita_Dhar_House_Price_Prediction
📌 Project Description

This project is a Machine Learning-based House Price Prediction Web App built using Streamlit. It allows users to estimate house prices based on key features such as number of bedrooms, bathrooms, living area, house condition, and nearby schools.

The model used in this project has been trained using a Random Search optimized algorithm, which improves prediction accuracy by tuning hyperparameters efficiently.

▶️ How to Run the Project
Clone the repository:
git clone https://github.com/your-username/Akshita_Dhar_House_Price_Prediction.git
Navigate to the project folder:
cd Akshita_Dhar_House_Price_Prediction
Install dependencies:
pip install -r requirements.txt
Run the app:
streamlit run app.py
📥 Input Parameters
The model takes the following inputs:
Number of Bedrooms
Number of Bathrooms
Living Area (sq ft)
Condition of the House
Number of Nearby Schools
📈 Output
Displays the predicted house price based on the input values.
